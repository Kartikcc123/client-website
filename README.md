 🚀 Getting Started

 📥 Clone Repository

```bash
git clone https://github.com/Kartikcc123/client-website.git
cd client-website
```

 ⚙️ Backend Setup

```bash
cd backend
npm install
node index.js
```

Backend will run on:

```bash
http://localhost:5000
```

 💻 Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```bash
http://localhost:5173
```

 🔐 Environment Variables

Create a `.env` file inside backend folder.

```env
PORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/academic-plus
JWT_SECRET=your_secret_key
```

 🌱 Database Seeding

Run this command inside backend folder:

```bash
node seed.js
```
